
### 🛠️ 核心技能

* **芯片方案：** 精通 **ESP32 系列**（ESP32-S3/C3）、熟练使用 Arduino/Uno R3。
* **硬件接口：** 熟练掌握 **I2C、SPI、UART、串口协议、PWM**（控制舵机/电机）。
* **模块应用：** 熟悉 GPS 模块、温湿度/空气质量传感器、NFC 触碰控制、墨水屏、各类驱动芯片。
* **多媒体处理：** 具备图片传输、视频传输搭建经验。
* **系统与平台：** 熟练配置 **Home Assistant**（智能家居集成）、PVE 系统环境搭建、Linux、CMS 搭建。

### 🚀 项目经历

#### 1. 自适应电脑亮度调节系统（实用性强）

* **技术栈：** ESP32 + 光敏传感 + 电脑端通信脚本
* **项目描述：** 硬件端实时采集环境光强数据，通过通信协议发送至 PC，软件端根据光强变化自动调节屏幕亮度。
* **成果：** 成功解决手动调节亮度的繁琐，已作为日常生产力工具长期使用。。

#### 2. IoT 智能门控系统（NFC + Web 控制）

* **技术栈：** ESP32 + NFC 模块 + 舵机 + Web 交互
* **项目描述：** 结合 Web 后台与物理 NFC 标签，实现多维度开门控制。支持手机 NFC 触碰即开，也支持远程网页发送指令。
* **成果：** 该项目已在家中部署并长期服役，实现了传统家居的智能化改造。

#### 3. 全能环境监测与智能家居集成

* **技术栈：** 温湿度传感器 + 空气质量/甲醛传感器 + Home Assistant
* **项目描述：** 搭建环境监测面板，实时展示空气质量。后期通过 MQTT/API 成功接入 **Home Assistant**，实现全屋环境自动化监测。

#### 4. 基于串口协议的自定义游戏控制板

* **技术栈：** Uno R3 + 按键板 + 摇杆 + 串口协议
* **项目描述：** 自定义摇杆与按键逻辑，通过串口协议模拟标准的 HID 游戏输入设备，让电脑识别为外部控制器。
